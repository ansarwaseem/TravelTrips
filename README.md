# TravelTrips

A modern travel and tourism website showcasing travel packages and destinations.

## Project Files

- **index.html** - Home page with hero section and featured destinations
- **about.html** - About us page with company information
- **contact.html** - Contact form for inquiries
- **Trips.html** - Travel packages and promotions listing
- **style.css** - Main stylesheet for the entire project
- **Images** - hero.webp, fig1.webp, fig2.webp, fig3.webp, fig4.webp

## Features

- Responsive navigation header with sticky positioning
- Hero sections with background images
- Travel package cards with pricing
- Contact form
- Social media links
- Smooth scrolling effects
- Modern CSS styling with animations

## How to Use

1. Clone the repository
2. Add your images (hero.webp, fig1.webp, fig2.webp, fig3.webp, fig4.webp) to the project folder
3. Open `index.html` in a web browser

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons (CDN)
- Google Fonts (Raleway)
